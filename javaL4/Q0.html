A for loop in Java is a special way to repeat a block of code a certain number of times. It's like saying, "I want to do this action over and over until I reach a specific number." For example, if you want to count from 1 to 5, you can use a for loop to help you do it quickly!

Here’s how a for loop works:

  • First, you start counting from a number (usually 0 or 1).
  • Then, you say how long you want to count (like up to 5).
  • Finally, you tell it how to count (usually by adding 1 each time).

This is what it looks like in Java code:

for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

Let’s break it down:

  • int i = 1; means we start counting from 1.
  • i <= 5; means we will keep counting as long as our number is less than or equal to 5.
  • i++ means that each time we count, we add 1 to our number.

When we run this code, it will print the numbers 1, 2, 3, 4, and 5. This makes for loops great for tasks like counting, going through lists of things, or repeating actions quickly without having to write the same code over and over again.
