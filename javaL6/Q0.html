In Java, an enhanced for loop (also known as a for-each loop) helps you easily look at every item in a group, like a list of toys or a set of numbers. You can use this loop when you want to check or do something with each item without needing to know where it is in the list. To create an enhanced for loop, you write it like this: for (type item : collection), where type is what kind of item youâ€™re looking at (like a number or a word), and collection is the list or group of things you want to go through. This loop will automatically show you each item one by one, making your code simpler and easier to read.

For example, if you have a list of numbers and you want to show each one, you can do it like this:

int[] numbers = {1, 2, 3, 4, 5};

for (int number : numbers) {
    System.out.println(number);
}

This code will print out each number in the list, one after the other.
