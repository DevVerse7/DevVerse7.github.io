String variables are used to store text, and this text is always enclosed in double quotes. It's important to remember that every statement should end with a semicolon (;). To display text to the user, you can use System.out.println();, which prints the text and moves the cursor to a new line, or System.out.print();, which prints the text on the same line. If you want to start a new line within your text, simply use \n within the quotations of your text.

You can also concatenate strings with other strings using the + operator. For example:

System.out.println("Hello, " + "world!");

This will output: Hello, world!.

Additionally, you can combine strings with other data types, such as integers and doubles, using concatenation. This is done by placing the + operator between the strings and the values. For example, if you want to display a message with a name and an age, you can write:

System.out.println("My name is Alice and I am " + 25 + " years old.");

This will output: My name is Alice and I am 25 years old.

You can also add integers and doubles directly in your print statements. For example:

System.out.println("The total is: " + (10 + 20 + 15.5));

This will output: The total is: 45.5. When adding numbers in a string context, it's important to use parentheses to ensure the addition occurs before the concatenation, allowing for accurate calculations in your output.
