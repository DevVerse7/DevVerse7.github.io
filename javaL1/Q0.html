Variables are special boxes in a computer program used to store information, each with a name for easy access. They help us keep track of different types of data, like numbers and words, making our programs smarter and more flexible. By allowing us to organize and manage information, variables enable us to build programs that solve problems and perform calculations. Without them, programming would be confusing and difficult. In short, variables are essential tools that make it possible to create effective and dynamic applications.

There are different types of variables in Java:

  • String: For words and sentences. Use double quotes.
    Example: String greeting = "Hello";
  • int: For whole numbers, like 1, 2, or 100. No quotes.
    Example: int age = 10;
  • double: For numbers with parts, like 3.14 or 2.5. No quotes.
    Example: double price = 5.99;
  • char: For a single letter or symbol. Use single quotes.
    Example: char letter = 'A';
  • boolean: For yes or no answers. You can say either true or false.
    Example: boolean isHappy = true;
  • final: If you want to create a variable that cannot change, use the word final before the type.
    Example: final int daysInWeek = 7; (This means it will always be 7!)
