In Java, a nested loop is like having one loop inside another loop. This is useful when you need to repeat tasks multiple times, such as looking at rows and columns in a table.

Hereâ€™s how it works:

  • Outer Loop: This is the first loop that goes through each row.
  • Inner Loop: This loop goes through each column inside the row.

Think of it like counting in a grid! For example, if we want to print numbers in a 3x3 grid, we can use nested loops like this:

for (int i = 1; i <= 3; i++) {        // Outer loop for rows
    for (int j = 1; j <= 3; j++) {    // Inner loop for columns
        System.out.print(j + " ");    // Print the column number
    }
    System.out.println();            // Move to the next line after each row
}

When we run this code, it prints:

1 2 3 
1 2 3 
1 2 3

This means for each row, we printed the numbers 1, 2, and 3, three times! Nested loops help us do things that need more than one step, just like counting rows and columns in a table.
